# INEAN PLAY OTOKEY-F Foldable Automatic OKEY Table

OTOKEY-F combines automatic OKEY tile handling with a foldable pedestal design. It is intended for homes, apartments, multi-purpose rooms, and other locations where storage flexibility matters.

Supported configurations provide standard OKEY and 101 OKEY modes, control-panel operation, and fault-code diagnostics. The foldable structure and casters can make storage and repositioning easier, subject to the confirmed product dimensions, weight, floor conditions, and safe handling procedure.

Before choosing OTOKEY-F, measure the playing position, folded storage area, doorway and lift access, and power-outlet location. Confirm the approved magnetic tile set and current installation instructions with INEAN PLAY.
