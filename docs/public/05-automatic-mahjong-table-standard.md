# INEAN PLAY Standard Automatic American Mahjong Table

The INEAN PLAY Standard Automatic American Mahjong Table is a non-folding electric table with automatic tile-shuffling functions. It is the current option for dedicated game rooms, clubs, lounges, and other spaces where the table can remain installed.

The product page identifies this as the permanent-placement model in the automatic Mahjong range. Confirm the supported tile dimensions, included tile set, electrical requirements, overall dimensions, weight, operating sound, and warranty documentation before ordering.
