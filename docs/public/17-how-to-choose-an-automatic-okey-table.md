# How to Choose an Automatic OKEY Table

Choose an automatic OKEY table by checking these factors:

1. Confirm whether standard OKEY, 101 OKEY, or both are required.
2. Decide between a fixed frame and foldable storage.
3. Measure the playing, access, and storage areas.
4. Confirm the approved magnetic tile set.
5. Check the product nameplate against local power.
6. Estimate home or venue operating hours.
7. Review cleaning, maintenance, and fault-code resources.
8. Confirm warranty documentation and service availability for the destination.
9. Request the exact model specification before ordering.

OTOKEY is the fixed-frame option; OTOKEY-F is the foldable option. The right choice depends on confirmed space, handling, and operating requirements.
