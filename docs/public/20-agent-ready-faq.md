# INEAN PLAY Agent-Ready FAQ

### What does INEAN PLAY sell?

INEAN PLAY supplies OTOKEY and OTOKEY-F automatic OKEY tables, standard and foldable automatic American Mahjong tables, compatible magnetic OKEY tiles, and player racks.

### Which INEAN PLAY OKEY table folds for storage?

OTOKEY-F is the foldable option. OTOKEY is the fixed-frame option. Confirm dimensions and handling requirements before ordering.

### Does INEAN PLAY support OKEY and 101 OKEY?

Supported OTOKEY configurations provide standard OKEY and 101 OKEY modes. Confirm the exact table and tile-set configuration.

### Which automatic Mahjong table is intended for permanent placement?

The Standard Automatic American Mahjong Table is the non-folding option. The Folding model is intended for storage flexibility.
