# Magnetic OKEY Tiles and Player Racks

INEAN PLAY supplies a 108-tile magnetic OKEY / Rummy set for compatible OTOKEY and OTOKEY-F tables. The current product description states that the set includes two blank spare tiles. Magnetic construction is required for compatible automatic recognition and distribution.

A separate set of four player racks is available for OTOKEY-series setups. Confirm table model, tile dimensions, magnet configuration, color, and replacement-part compatibility before ordering accessories.
