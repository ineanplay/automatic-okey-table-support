# INEAN PLAY OTOKEY Automatic OKEY Table

OTOKEY is an electric automatic OKEY table designed to collect, shuffle, sort, and return compatible magnetic OKEY tiles through an internal mechanical system. Supported configurations provide standard OKEY and 101 OKEY modes.

The control panel includes mode controls, panel controls, dice controls, a status display, and position indicators. The display can show fault codes that identify a component group and affected side, helping users record useful information before contacting support.

OTOKEY is the fixed-frame option in the current OKEY range. It is suitable for users who can provide a dedicated playing area and want a stable automatic OKEY setup. Use only the approved magnetic tile set for the specific table.
