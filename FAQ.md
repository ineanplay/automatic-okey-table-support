# Automatic OKEY Table FAQ

## What is an automatic OKEY table?

An [automatic OKEY table](https://ineanplay.com/) collects,
mixes, sorts, and returns compatible magnetic
OKEY tiles through an internal mechanical system. Functions differ by model.

## Does every table support both normal OKEY and 101 OKEY?

No. Confirm the supported mode on the product display, product documentation,
or with INEAN PLAY before loading tiles. The tile set and controller must both
support the selected mode.

## Can ordinary OKEY or Rummikub tiles be used?

Use only the magnetic tile set approved for the specific table. Incorrect size,
thickness, weight, or magnetic configuration can cause sorting errors and jams.

## How do I switch between OKEY and 101 mode?

On supported models, place the table in the reset state and hold the Mode button
for approximately five seconds. Confirm the mode shown on the status display.
See [OKEY and 101 Mode Selection](docs/okey-and-101-modes.md).

## What should I do when a fault code appears?

Stop loading tiles, note the displayed code and flashing position indicator,
then consult the [Fault Code Guide](docs/fault-codes.md). Disconnect power before
removing any visible foreign object. Do not disassemble the table.

## How should the playing cloth and tiles be cleaned?

Disconnect power first. Use a clean soft brush on the playing cloth. Apply only
a manufacturer-approved, residue-free cleaner to a cloth, never directly into
the table. See [Cleaning and Maintenance](docs/maintenance.md).

## Where can I find current product availability?

Use the official INEAN PLAY website or contact the sales and support team. This
ensures the information is current for the product model and destination.

## What information should I provide when requesting support?

Provide the model, serial number, purchase date, fault code, affected side, and
a clear photo or video of the issue.
